Politique de Confidentialité

← Retour à l'accueil
Dernière mise à jour : 30 juillet 2025

1. Introduction

Cette politique de confidentialité décrit comment StreamPilot ("nous", "notre" ou "l'extension") collecte, utilise et protège vos informations lorsque vous utilisez notre extension Chrome.

StreamPilot est conçu pour améliorer votre expérience de visionnage vidéo sur le web tout en respectant votre vie privée.

2. Informations que nous collectons

2.1 Données collectées localement

StreamPilot collecte et stocke uniquement les informations suivantes sur votre appareil local :

  • Préférences utilisateur : Vos paramètres de vitesse de lecture préférés
  • État des contrôles : Statut du mode Picture-in-Picture et du son
  • Paramètres de l'extension : Configuration de l'interface utilisateur
Important : Toutes ces données sont stockées localement dans votre navigateur et ne sont jamais transmises à des serveurs externes.

2.2 Données que nous ne collectons PAS

StreamPilot ne collecte, ne stocke et ne transmet aucune des informations suivantes :

  • Informations personnelles identifiables
  • Historique de navigation
  • Contenu des vidéos regardées
  • URLs des sites visités
  • Données de géolocalisation
  • Informations de connexion ou mots de passe

3. Comment nous utilisons vos informations

Les données collectées localement sont utilisées exclusivement pour :

  • Sauvegarder vos préférences entre les sessions
  • Améliorer l'expérience utilisateur de l'extension
  • Maintenir l'état des contrôles vidéo

4. Partage des données

StreamPilot ne partage aucune donnée avec des tiers. L'extension fonctionne entièrement en local et n'établit aucune connexion avec des serveurs externes.

5. Sécurité des données

Vos données sont protégées par les mécanismes de sécurité intégrés de Chrome :

  • Stockage local sécurisé via l'API Chrome Storage
  • Isolation des données par extension
  • Chiffrement automatique des données stockées

6. Permissions demandées

StreamPilot demande les permissions suivantes pour fonctionner :

  • activeTab : Pour interagir avec l'onglet actif
  • storage : Pour sauvegarder vos préférences localement
  • scripting : Pour injecter les contrôles vidéo
  • contextMenus : Pour ajouter des options au menu contextuel
  • tabs : Pour gérer les onglets avec des vidéos

7. Vos droits

Vous avez le contrôle total sur vos données :

  • Accès : Toutes vos données sont stockées localement dans votre navigateur
  • Modification : Vous pouvez modifier vos préférences à tout moment
  • Suppression : Désinstaller l'extension supprime automatiquement toutes les données
  • Portabilité : Vos préférences sont liées à votre profil Chrome

8. Conservation des données

Les données sont conservées uniquement tant que l'extension est installée. La désinstallation de StreamPilot supprime automatiquement toutes les données associées.

9. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité occasionnellement. Les modifications seront publiées sur cette page avec une nouvelle date de "dernière mise à jour".

10. Contact

Si vous avez des questions concernant cette politique de confidentialité ou le fonctionnement de StreamPilot, vous pouvez nous contacter via GitHub.

Résumé : StreamPilot respecte votre vie privée en ne collectant aucune donnée personnelle et en fonctionnant entièrement en local sur votre appareil.
